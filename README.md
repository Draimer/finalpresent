# 互動 HTML 簡報

開啟 `index.html` 即可播放。整份簡報使用 1920x1080 畫布，原始投影片圖片保留作為精準對位底圖。

## 圖片替換

把替換圖片放在：

`assets/parts/`

檔名可用簡報上的標籤，例如：

- `P28p1.jpg`
- `P28p2.png`
- `P29p3.jpg`
- `QR.jpg`

重新開啟 `index.html` 後，對應的圖片區會自動載入。也可以在 HTML 內按「編輯」，雙擊圖片區或按「換圖」臨時替換。

## 互動功能

- 換頁動畫：上一頁會淡出滑開，新頁會淡入滑入。
- 頁面進度：上方有酸綠進度條與頁點，可直接點頁點跳到指定頁。
- 紅框聚焦：播放或編輯時點擊任何可互動元素，該元素會被紅框標示；點空白處可清除。
- OCR 文字層：每頁辨識出的字/詞都已疊成透明文字元素，可框選與複製；點擊單一 OCR 字時，紅框會自動框住同一行、相鄰的一組文字。
- 簡報工具：右下角可切換雷射、聚光燈、紅筆標註與清除筆跡。快捷鍵：`L` 雷射、`S` 聚光、`P` 紅筆、`C` 清除、`F` 全螢幕播放。
- QR / 加入互動：上方 `QR` 按鈕可貼互動連結，也可上傳自訂 QR 圖，會保存在本機瀏覽器。
- 投票：可現場按選項累加票數並顯示比例。
- 倒數：可用於分組討論或現場活動。
- 問答：可把台下問題放大顯示。
- 抽籤：可輸入名單並隨機抽出。

## 編輯狀態

瀏覽器會自動保存本機編輯狀態。需要搬移到別台電腦時，先按「匯出」下載 JSON，再到另一台電腦按「匯入」。
